# Clist [![Licence](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

### Description
Clist is a TUI note managing app made in c++ and sqlite3 as a studying project.

### Features
- Adding tasks
- Deleting tasks
- Editing tasks
- Vim like keybindings (In progress...)
- Completing tasks (In progress...)

### Build Instructions

```bash
$ git clone https://github.com/Chupchushka/Clist.git
$ cd Clist
$ make 
$ ./build/waybar
```

### Uninstalling
``` bash
$ make clean
```
